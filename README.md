# DeskGuard-AI
智能屏幕隐私保护助手 | 自动识别工作/摸鱼状态 + 有人靠近自动隐藏

---

## ✨ 功能介绍
- 摄像头实时检测身边人数
- 屏幕OCR识别当前工作/娱乐状态
- DeepSeek AI 智能决策
- 摸鱼模式 + 有人靠近 → 自动隐藏屏幕
- 普通模式：人数≥2 → 自动保护
- 可自定义检测频率（秒）
- 无API Key自动降级，不影响使用

## 🚀 运行方式
```bash
python ui_panel.py
```

## 🎯 运行逻辑
- AI 模式：人数≥2 + 摸鱼 → 隐藏
- 普通模式：人数≥2 → 隐藏
- 无 API Key：自动使用普通模式

## ⚙️ 配置文件
- config.py → AI 开关、检测间隔
- settings.json → DeepSeek API Key

---

## 💰 运行成本估算 (Daily Cost)
该项目在设计时充分考虑了 **LLM 推理成本** 与 **边缘侧资源占用**：

* **API 消耗：** 采用 DeepSeek-V3/R1 接口，单次决策 Token 数约 150-200。若设定检测频率为 30s/次，工作 8 小时约调用 960 次。
    * **日均成本：** 按照当前 DeepSeek 费率，约 **￥0.05 - ￥0.1 / 天**（极低成本，适合长期后台挂载）。
* **本地资源：**
    * **CPU 占用：** < 5% (基于 OpenCV 的轻量级人脸检测)。
    * **内存占用：** ~150MB (Python 运行时 + 基础视觉库)。
